# Firebase-Auth

> Firebase Authentication + Firestore 기반 사용자 인증 및 데이터 영속화 모듈 (GRID-SaaS V16).

## 설명

Firebase SDK 3종(app-compat, auth-compat, firestore-compat) 로드. 사용자 로그인 상태에 따라 사이드바 프로필, "My Projects", "Recent History" 기능 활성화. 분석 결과 및 설정을 Firestore에 저장해 세션 간 유지.

## 기술 스택

- **firebase-app-compat 10.7.1**
- **firebase-auth-compat 10.7.1** — Google OAuth / 이메일 인증
- **firebase-firestore-compat 10.7.1** — 사용자 데이터, 프로젝트, 히스토리
