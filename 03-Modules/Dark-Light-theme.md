# Dark-Light-theme

> Dark / Light 테마 전환 모듈. CSS Custom Properties 기반, `toggleDarkMode()` 함수.

## 설명

CSS 변수(`--bg-app`, `--text-main`, `--accent` 등) 전체를 `.light-mode` 클래스 토글로 교체. 사용자 선호 또는 OS 설정(`prefers-color-scheme`)에 따라 자동 전환. 선택값을 `localStorage`에 저장해 세션 간 유지.

## 기술 스택

- **CSS Custom Properties** — 색상 팔레트 변수
- **JavaScript** (테마 토글 로직, `toggleDarkMode()`)
- `localStorage` — 사용자 설정 유지
