# SaaS-Dashboard

> GRID-SaaS의 메인 대시보드 UI 모듈.

## 설명

사용자가 이미지 업로드, Gemini 분석 요청, 차트 확인, 리포트 다운로드를 수행하는 통합 인터페이스.

## 기술 스택

- **HTML / CSS / JavaScript**
- Chart.js (차트 렌더링)
- Fetch API (프록시 통신)
