# Express-server

> Node.js Express.js 기반 HTTP 서버 코어 모듈.

## 설명

포트 3000에서 실행되는 Express.js 서버. 정적 파일 서빙, API 라우팅, CORS 처리, 파일 업로드(50MB 한도) 담당.

## 기술 스택

- **Node.js**
- **Express.js**
- **cors**, **dotenv**, **multer** (또는 express-fileupload)

## 설정

```
PORT=3000
GEMINI_API_KEY=<.env에서 관리>
```
