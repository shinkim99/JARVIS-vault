# FastAPI-backend

> Python FastAPI 기반 REST API + SSE 엔드포인트 모듈.

## 설명

FastAPI로 구현된 비동기 백엔드. REST 엔드포인트와 Server-Sent Events(SSE) 스트리밍 엔드포인트를 모두 제공.

## 기술 스택

- **Python** + **FastAPI** + **Uvicorn**
- 비동기 처리: `async`/`await`
- SSE: `EventSourceResponse`
