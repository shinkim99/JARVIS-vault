# Flowchart-UI

> 파이프라인 단계를 실시간으로 시각화하는 플로우차트 UI 모듈.

## 설명

SSE 이벤트를 수신하여 파이프라인 각 단계의 상태(대기·진행·완료·오류)를 플로우차트로 렌더링. Step 간 화살표 및 상태 색상 표시.

## 기술 스택

- **JavaScript** (동적 DOM 조작)
- **SVG** (화살표 렌더링)
- **HTML/CSS**

## 알려진 이슈

- Step 1→2 화살표 렌더링 오류 (진행 중)
