# AES-Encryption

> 비밀번호 기반 AES 암호화/복호화 모듈 (wb-library).

## 설명

wb-library의 핵심 보안 레이어. 라이브러리 데이터를 AES로 암호화해 로컬에 저장. 비밀번호 없이는 내용에 접근 불가. NEXUS vault 통합 전 단독 운영되던 모듈.

## 기술 스택

- **JavaScript Web Crypto API** (또는 CryptoJS)
- **localStorage** — 암호화된 데이터 저장
