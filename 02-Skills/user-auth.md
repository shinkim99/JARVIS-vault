# user-auth

> Firebase Authentication 기반 사용자 인증. Google OAuth + Firestore 사용자 데이터 연동.

## 설명

GRID-SaaS V16에서 Firebase SDK(firebase-app-compat, firebase-auth-compat, firebase-firestore-compat)로 구현. 사용자 프로필, 프로젝트 히스토리를 Firestore에 영속화.

## 사용 프로그램

- [[GRID-SaaS]] — 사이드바 프로필, "My Projects", "Recent History" 기능
