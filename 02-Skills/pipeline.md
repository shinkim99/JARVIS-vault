# pipeline

> 멀티스테이지 처리 파이프라인 설계 및 실행. 단계 간 상태 전달, 오류 처리, 중간 결과 스트리밍.

## 설명

입력 → 여러 처리 단계 → 최종 출력으로 이어지는 파이프라인을 구성하고 실행. 각 단계의 입출력을 관리하고 SSE로 진행 상황을 실시간 전달.
