# n8n-webhook

> n8n 클라우드 워크플로우를 웹훅 POST로 트리거하고 결과를 수신.

## 설명

`https://shinkim99.app.n8n.cloud/webhook/tech-intel-run`으로 도메인 설정(JSON)을 POST. n8n 워크플로우가 분석 수행 후 결과를 postMessage 또는 URL 파라미터로 반환. LocalStorage로 결과 영속화.

## 사용 프로그램

- [[tech-intelligence]] — 기술 인텔리전스 도메인 분석 트리거
