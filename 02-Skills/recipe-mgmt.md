# recipe-mgmt

> 전극 슬러리 레시피 관리. 배합비 계산, 런시트 생성, 버전 관리.

## 설명

음극/양극 슬러리 제조 레시피를 표준화하고, 성분 배합비를 자동 계산하여 런시트 형태로 출력. 히스토리 추적 기능 포함.
