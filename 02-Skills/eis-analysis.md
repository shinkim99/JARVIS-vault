# eis-analysis

> 전기화학 임피던스 분광법(EIS) 데이터 분석. Nyquist·Bode 플롯, 등가회로 피팅.

## 설명

EIS 장비(Gamry, BioLogic 등) 출력 데이터를 분석. Nyquist 플롯에서 계면 저항(Rct), 확산 임피던스(Zw) 등을 추출.
