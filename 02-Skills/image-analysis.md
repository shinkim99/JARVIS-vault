# image-analysis

> 이미지 분석 및 특징 추출. AI Vision API 기반 객체 인식, 텍스트 추출(OCR), 패턴 분석.

## 설명

업로드된 이미지를 AI 모델에 전달하여 분석 결과를 구조화. 배터리 전극 SEM, 분산 이미지 등 연구용 이미지 분석에 특화.
